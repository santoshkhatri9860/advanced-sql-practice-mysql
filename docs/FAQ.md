# FAQ

## Do I need Docker?
No. Docker is optional. You can run MySQL locally if you prefer.

## Is this only for beginners?
No. It starts simple, but quickly goes into advanced patterns:
CTEs, recursive CTEs, window functions, and date/time logic.

## Are the solutions the only correct answers?
Not always. SQL often has multiple correct solutions.
The provided solutions are reference implementations.

## Can I use this for interview prep?
Yes. This repo is designed to build pattern recognition through repetition.
